# Document-Intelligent

Azure AI Document Intelligence is a cloud-based AI service that extracts text and important data from documents, PDFs, and images using OCR technology.

## Features
- Extract text from PDFs and images  
- Analyze documents using AI  
- User-friendly UI/UX website  
- Built with Python and Streamlit  
- Uses Azure AI Document Intelligence API  

## Technologies Used
- Python  
- Streamlit  
- Azure AI Document Intelligence  
- OCR Technology  

## How It Works
1. Upload a document or image  
2. Click on Start Analysis  
3. Azure AI processes the file  
4. Extracted text is displayed  
5. Download the result as a text file  

## Purpose
This project helps users quickly extract and analyze text from documents using cloud-based AI services.
